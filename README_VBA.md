# UPH 流水线计算器 - Excel VBA 版本

## 安装说明

### 方法一：直接导入模块

1. 打开 Excel，新建或打开一个工作簿
2. 按 `Alt + F11` 打开 VBA 编辑器
3. 在左侧项目窗口中，右键点击 `VBAProject (你的工作簿名)`
4. 选择 `导入文件` → 选择 `UPHCalculator.bas`
5. 关闭 VBA 编辑器
6. 按 `Alt + F8`，选择 `CreateUPHCalculator`，点击 `运行`

### 方法二：手动创建

1. 按 `Alt + F11` 打开 VBA 编辑器
2. 右键点击项目 → `插入` → `模块`
3. 将 `UPHCalculator.bas` 的全部内容复制粘贴到模块中
4. 关闭 VBA 编辑器
5. 按 `Alt + F8`，运行 `CreateUPHCalculator`

## 功能说明

### 配置管理
- **保存** - 保存当前工站配置
- **另存为** - 保存为新配置名称
- **删除** - 删除当前配置（至少保留一个）
- **下拉菜单** - 切换不同配置

### 工站配置
- 工站名称：自定义名称
- 处理时间：每个产品在该工站的处理时间（分钟）
- 缓冲容量：工站之间的缓冲区容量

### 计算结果
- **实际 UPH** - 基于模拟的实际产能（件/小时）
- **总耗时** - 所有产品完成的总时间
- **理论 UPH** - 瓶颈工站限制下的理论最大值
- **瓶颈工站** - 利用率最高的工站
- **工站利用率** - 每个工站的忙碌时间占比
- **时间线事件** - 产品在各工站的处理时间线

## 算法说明

采用事件驱动的流水线模拟，正确处理**阻塞反压**：

1. 当下游缓冲区满时，上游工站被阻塞
2. 被阻塞的产品在当前工站输出端等待
3. 当下游缓冲区有空位时，被阻塞产品进入，工站解除阻塞

## 注意事项

- 最多支持 10 个工站
- 配置数据保存在 VBA 内存中（关闭文件后丢失）
- 如需永久保存配置，可在 Excel 单元格中存储
